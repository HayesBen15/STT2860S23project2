Dataset #1: Disney+ Data

The Disney+ dataset contains a variable that gives information about the rating for each program offering. Convert that variable into a factor that puts the ratings in (roughly) ascending order of age restriction/audience: TV-Y, TV-Y7, TV-Y7-FV, TV-G, G, TV-PG, PG, PG-13, TV-14. Delete the show_id, director, cast, and description variables. You should be able to do this all in one piped sequence.

disneydata2 <- disneydata %>%
  mutate(rating = factor(rating, levels = c("TV-Y", "TV-Y7", "TV-Y7-FV", "TV-G", "G", "TV-PG", "PG", "PG-13", "TV-14"))) %>%
  arrange(rating)

Dataset #2: Streaming TV Data

Perform the following manipulations to tidy the streaming TV data for analysis. You should be able to do this all in one piped sequence.

  • Pivot the Netflix, Hulu, Prime Video, and Disney+ variables into two columns called Service (containing the name of the streaming service) and YesNo (containing the 0/1 values from the original columns).
  • Filter the dataset to remove any row where YesNo == 0 (i.e., a show is not on the service).
  • Separate the IMDb variable to split each shows actual rating from the maximum possible rating of 10.
  • Convert the shows actual IMDB rating into a numerical variable and get rid of the column containing 10.
  • Separate the Rotten Tomatoes variable to split each shows actual rating from the maximum possible rating of 100.
  • Convert the shows actual Rotten Tomatoes rating into a numerical variable and get rid of the column containing 100.
TidyStreamingData <- StreamingData %>%
  pivot_longer(
    cols = c("Netflix", "Hulu", "Prime.Video", "Disney."),
    names_to = "Service",
    values_to = "YesNo"
  ) %>%
  filter(YesNo > 0) %>%
  separate(col = IMDb, into = c("Actual_IMDb_Rating", "IMDb_Possible_Rating"), sep = "/") %>%
  separate(col = Rotten.Tomatoes, into = c("Actual_Rotten_Tomatoes_Rating", "Rotten_Tomatoes_Possible_Rating"), sep = "/") %>%
  mutate(
    Actual_IMDb_Rating = as.numeric(Actual_IMDb_Rating),
    IMDb_Possible_Rating = NULL,
    Actual_Rotten_Tomatoes_Rating = as.numeric(Actual_Rotten_Tomatoes_Rating),
    Rotten_Tomatoes_Possible_Rating = NULL
  )
